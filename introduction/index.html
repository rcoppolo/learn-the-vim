Learn to do the Vim.

What's this Vim thing anyway?

  • Vim is a text "editor"

    It shines at manipulating text, not just typing it in.

  • Vim is old

    It's an extension of Vi, which came out in 1976*.

  • Vim is keyboard based

    You will never even have to reach for the arrow keys, let alone the mouse.

  • Vim is everywhere

    You'll be right at home on remote servers.









* Bohemian Rhapsody also came out in 1976.

How to get started

  • Do vimtutor!

    Open a terminal and type "vimtutor".

  • "Learn Vim Progressively"

    1. Install Vim (not required).
    2. Launch Vim.
    3. DO NOTHING...

Learn Vim Progressively

This guy thinks you should start by learning the most essential commands and just plugging away.


Otherwise the vast number of keyboard shortcuts will overwhelm and discourage you.

So what are these "essential" commands?

You move around with h,j,k,l.*


i gets you into insert mode (where you can type).


ESC gets you out of insert mode.


:w saves your file.


:q quits. (You can combine the last two commands with :wq).


:help saves your life.










* Think of "j" as an arrow pointing down. Even though "j"'s do not look anything like arrows pointing down.

Baby steps...

These commands work in Vim's "normal" mode where you can't actually insert text:


u undos your last completed command.


x deletes the character under the cursor.


dd deletes the current line.


p pastes.


Basic movement...

0 goes to the first column of the line.


$ goes to the end of the line.


w moves forward one word (backwards is b).


A inserts at the end of the line (I does the same at the beginning).


G moves to the last line (gg goes to the first).


Fancy stuff...

PROTIP: You can prefix any command with a number and Vim will execute that command that many times.


For instance: 5dd deletes 5 lines.


o inserts a line below the cursor (O does the same thing but above the cursor).


yy copies the current line.


cw changes the word (automatically pops you into insert mode).


. repeats the last completed command.


This is only the begininng...

Here are some commands I learned in the process of creating this slideshow:


dit deletes everything inside the html tags that your cursor is inside.


t[any character] goes to just before that character (T does the same thing backwards).


Thus dt[any character] deletes everything up to that character.

Practice makes perfect

I've been learning Vim for several months, and I've still only scratched the surface.


But my productivity should skyrocket any day now.
